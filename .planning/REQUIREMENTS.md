## Active

### Platform & Architecture
- [ ] Monorepo-style or separate repos for frontend, backend, and python microservices.
- [ ] Dockerized local setup with Redis and MongoDB.

### Auth & User
- [ ] User registration, login, logout, refresh tokens (JWT).
- [ ] User profile and progress tracking over multiple sessions.

### Interview Flow
- [ ] Dashboard to start an interview session (Select HR/Tech/Mixed, Role, Count).
- [ ] Camera and microphone permission checks and preview.
- [ ] Live interview UI with AI TTS question playing.
- [ ] Record user answer (video + audio) and submit to backend.
- [ ] Writing test module with 10-minute timer.

### AI Microservices
- [ ] Face service (confidence, nervousness, attention, eye contact).
- [ ] Voice service (transcript, fluency, speaking speed).
- [ ] NLP service (relevance, structure, grammar).
- [ ] Question & Report generation service using GPT-4o / Claude.

### Results
- [ ] Processing loader UI.
- [ ] Detailed report with overall score, sub-scores, strengths, weaknesses, and actionable recommendations.

## Out of Scope
- [ ] Real human interviewers
