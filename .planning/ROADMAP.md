# Project Roadmap

## Phase 1: Project Setup

**Goal:** Initialize all repositories/folders, configure environments, and set up the foundation.

## Phase 2: Auth System

**Goal:** Implement full JWT authentication flow.

## Phase 3: Dashboard + Setup Page

**Goal:** Build the pre-interview flow and integrations.

## Phase 4: Live Interview Page

**Goal:** Build the core recording interface.

## Phase 5: AI Analysis Services

**Goal:** Build Python microservices and async job queue.

## Phase 6: Writing Test

**Goal:** Implement the written evaluation module.

## Phase 7: Report Generation

**Goal:** Aggregate analysis and generate final reports.

## Phase 8: History + Profile + Progress

**Goal:** Build user tracking pages.

## Phase 9: Polish + Error Handling

**Goal:** Make the application production-ready.

## Phase 10: Dockerize + Deploy

**Goal:** Containerize and deploy the application.
